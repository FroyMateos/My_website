    Transmisor FM con comunicación RDS

    Desarrollar e implementar en una sonoboya (combinación de sonar y boya) un transmisor para que proporcione datos de su entorno y que mediante ondas FM transmitirá esa información mediante RDS.

    La problemática era que al desplegar la sonoboya en el mar no se sabia con exactitud su ubicación, estado climático o el estado de la batería.

    Tareas desarrolladas

  • Investigación de cada módulo que se implementa en el transmisor.
  • Desarrollar la programación.
  • Desarrollar las pruebas de los módulos y observar su implementación.
  • Desarrollar la simulación del ensamblaje del componente.
  • Diseñar el PCB donde todo los componentes del transmisor irán ensamblados.

    Primera página Web

    Con la finalidad de comprender HTML y CSS, se optó por desarrollar la página web en la cual se encuentra navegando usted. El principal enfoque de esta página web es ser utilizada como un portafolio de evidencia de los trabajos más relevantes desarrollados.

    Para el desarrollo de esta página fue utilizado el micro framework Flask (con el lenguaje Python) por tener las herramientas necesarias para crear una aplicación funcional y también tener muchas exenciones para dotar de mas funcionalidades a Flask.

    Para poder subir la página web a la nube se eligió el servicio de Heroku ya que está a diferencia de otras plataformas permite desarrollar prácticamente con cualquier lenguaje de programación: Ruby, Java, PHP, NodeJS, Python. Al igual que cuenta con un servicio gratuito el cual podemos acceder a las prestaciones básicas.

    Nota: La información que se encuentra en ese sitio Web no está actualizado.
